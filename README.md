you can download DatabaseChat from : https://www.softwarefactoriesofamerica.com and selecting "ZIP Product"

Unzip the file and then:

Right Click on the executable "DatabaseChatSetup_v1.0.0.exe" and select "Properties"
if the "UnBlock" is NOT checked, check the "UnBlock" and then press "Apply" and "Ok"
Then you can run the executable "DatabaseChatSetup_v1.0.0.exe"

The installation will:
  a. Check if "PostgreSQL" is installed and if NOT installed, "PostgreSQL" will be installed
  b. then DatabaseChat will be installed.
    1. the database data is migrated automatically
  c. a desktop icon is installed on your desktop
  
When installed, you can run which will open the code
When running open your favorite browser and enter: "http://localhost:5005/"
  a. if the browser states that the "site is not found", then DatabaseChat is not running
  
the first thing you need to do is to select "settings"
  a. enter your email address
  b. enter your name
  c. retrieve and enter an OpenAPI license
  d. configure your database(s) that you want to query against using DatabaseChat
  
Be sure to select "AI Build" which will help you build Natural Language Database queries like:
  a. "What are my quarterly sales for 2026"
    1. this depends on your selected databases
